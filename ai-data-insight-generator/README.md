# AI Data Insight Generator

AI Data Insight Generator is an AI-powered analytics platform that transforms raw CSV and Excel datasets into meaningful insights, visualizations, summaries, and business recommendations.

The platform combines:
- Data Engineering
- AI-powered analytics
- Business intelligence
- Modern deployment systems
- AI-native workflows

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/ai-data-insight-generator.git
cd ai-data-insight-generator
```

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
